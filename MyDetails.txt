#git commands details

1. git init
2. git clone "repo url"
3. git branch "branch_name" //to create new branch
4. git branch //to show all branch
5. git checkout -b "branch_name" //create new branch & checkout instantly
6. git checkout "branch_name" // to checkout existing branch
7. git commit -m "your message" // to commit changes
8. git status //to check changes
9. git add . //to track/stages all changes
10. git push // to push changes/commits
11. git pull //to pull latest changes
12. git rebase main // to make main branch and child branch at same level
